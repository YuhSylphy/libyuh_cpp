libyuh_cpp
==========

自分用C++ライブラリ

- VSTest/  
    ユニットテストコード(CppUnitTestFramework, Visual Studio 2012)
- docs/  
    doxygenによるドキュメント．詳細はこちらで
- include/  
    ヘッダファイル群
- stc/  
    実装部ソース
- test/  
    ユニットテストコード(google test)
- vcxproj/  
    Visual Studio用ソリューションファイルほか

test以下のビルドにはtestと同じ階層にgtest-1.6.0が必要です．
